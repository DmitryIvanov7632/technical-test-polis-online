# Простой блог с комментариями

## Используемые технологии

Zustand, React Router, React Hook Form, Zod, Axios

## Функционал

- Просмотр списка статей
- Просмотр одной статьи с комментариями
- Добавление новой статьи
- Добавление комментариев к статье

## Дополнительно реализованный функционал

- Разделённый state менеджер и простое кэширование: Zustand хранит статьи и комментарии отдельно, данные кэшируются в состоянии, лишние GET-запросы не выполняются.
- Оптимизация: рендеры компонентов обновляются только при изменении данных, лишние API-запросы на сервер не выполняются.
- Форма добавления статьи вынесена в header и открывается по кнопке через модальное окно.
- Backend: GET `/api/articles/{id}/comments` — получение всех комментариев для конкретной статьи.
- Базовая валидация: реализована как на фронтенде (React Hook Form + Zod), так и на бэкенде (Laravel validation).

## Инструкция по запуску

1. Склонируйте репозиторий
2. Перейдите в папку проекта
3. Запустите команду:
   docker-compose up -d
4. После запуска контейнеров автоматически:
    - устанавливаются зависимости фронтенда (npm install)
    - запускается dev-сервер фронтенда (npm run dev)
    - выполняются миграции и сидеры
5. Приложение будет доступно по адресу:
   http://localhost:5173
